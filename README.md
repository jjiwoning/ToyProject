# ToyProject

---
회원가입, 게시판 기능이 있는 프로젝트입니다.  

Session을 활용하여 로그인 기능을 구현   

Querydsl을 활용하여 게시판 페이징 동적 쿼리 구현   

Spring Interceptor를 활용하여 로그인 체크 구현   


개발 툴
- 언어: Java 11
- 프레임워크: Spring
- 도구: Spring Boot 2.7.2, Spring Data JPA 2.2.3 , Lombok, querydsl
- 데이터베이스: H2 Database 1.4.200
